# Sales Nav Scraper

Node.js + Selenium dashboard for Sales Navigator scraping jobs.

## Setup

- Place Chrome extensions in extensions/contactout and extensions/lusha
- Paste LinkedIn cookies JSON in the UI and save
- Update contactout.json and lusha.json with cookies if needed

## Scripts

- npm run dev
- npm start
